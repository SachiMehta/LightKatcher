# LightKatcher

**LightKatcher** is a pair of auto-dimming sunglasses that is designed to block the wearer from flashing lights. This could help individuals with photosensitive epilepsy where flashing lights are often a trigger for seizures.

## ✨ Features

- Quick response to flashing lights  
- Adjustible light-level
- Custom 3D-printed enclosure  
- Low-cost build (under \$30)

## 🛠️ Hardware & Electronics

- Microcontroller: Arduino Nano  
- LCD light valve
- Photodiodes    
- Custom-designed electronics and wiring  
- 3D-printed casing for durability and comfort
